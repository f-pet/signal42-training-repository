# Homework 01 — Build Something with AI

## The Challenge

Build something entirely through AI — no hand-coding allowed.

Use any tool, any language, any idea. The only constraint: let AI do the coding, and you do the thinking.

Post your result in this folder under your own name or alias (e.g. `homework-01/your-name/`), and push it to the repository before Session 2.

Standout projects will be recognized at the start of **Session 2: Prompting & Context**.
